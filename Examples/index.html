HTML-in-Canvas demos

Live examples of the WICG HTML-in-Canvas proposal — draw real, interactive, accessible HTML into 2D, WebGL, and WebGPU canvases.

Flag required. These demos only run in Chromium Canary with chrome://flags/#canvas-draw-element set to Enabled.

Your browser:

Starter demos

The original WICG examples. Each exercises one API surface cleanly.

2D

Complex text

Rotated multi-line text with emoji, RTL, vertical CJK, inline SVG and images.
2D

Pie chart

Focusable, keyboard-navigable pie chart with multi-line HTML labels.
2D

Text input

A full interactive form drawn into canvas with keyboard, focus, and IME.
WebGL

HTML on 3D cube

HTML rendered as a texture on a spinning 3D cube via texElementImage2D.
WebGPU

Jelly slider

Range input under a WebGPU jelly-physics distortion shader (bundled project).

Advanced demos

New demos exercising what overlays, html2canvas, and SVG foreignObject can't.

WebGL · single file

Holographic card

Pokémon-style holographic foil over a real HTML business card with working email, phone, and URL links. Mouse-tracked iridescence.
2D · single file

Infinite canvas

Figma-style zoomable workspace. Six live widgets (form, sparkline, calc, typography). Drag to pan, scroll to zoom, Alt+drag to move a panel.
WebGL + 2D · single file

Trading terminal

Bloomberg-style live ticker table through a CRT/bloom/scanline shader. Sortable, focusable, accessible rows.
WebGPU · bundled

Liquid-glass nav

Apple-style liquid-glass navigation bar with chromatic dispersion refracting a scrollable article underneath. Real focusable links. Requires npm install && npm run dev.
WebGPU · bundled

Crystal hands

Webcam hand tracking (MediaPipe) + WebGPU SDF ray-march renders your hands as translucent crystal with caustics. Live HTML stats HUD via copyElementImageToTexture. Requires npm install && npm run dev.
WebGPU · bundled

Shadow puppet

Hold a hand gesture up to your webcam — 8 gestures trigger cute cartoon animals that animate across the screen. Three-canvas stack: WebGPU particle background, 2D animals, HTML-in-Canvas HUD. Requires npm install && npm run dev.
